# PID 控制培训代码仓库

## 项目简介

这是一个基于 STM32F407VE 的 PID 控制培训代码仓库，由仓库所有者创建，用于展示工作内容与过程。

## 硬件平台

- **主控芯片**: STM32F407VET6
- **电机类型**: 大疆 RoboMaster M3508 减速电机
- **电调类型**: 大疆 RoboMaster C620 无刷电调
- **通信接口**: CAN 总线通信

## 主要功能

### 1. PID 控制器
- 实现位置环、速度环双环PID控制
- 支持串级PID控制（位置-速度串级）
- 支持前馈控制，提高系统响应速度
- 可配置的积分分离、积分限幅、微分先行

### 2. 电机控制
- CAN 总线通信，支持多电机控制
- 编码器数据处理，支持多圈角度计算
- 减速比转换，输出轴角度计算
- 编码器回绕边界处理，确保角度连续性

### 3. 控制模式
- **单速度环模式**: 速度闭环控制
- **单位置环模式**: 位置闭环控制
- **串级模式**: 位置-速度串级控制
